# C｜地涌生成型

适用于：主题核心是“某种物质、结构、能量体或意象从地面内部向上生成、破土、裂出、突生、喷涌、隆起或蔓生”的技能。重点不是静态领域本体，不是从高处垂直降下的裁决，也不是多个打击体连续覆盖，更不是轨迹性穿行事件，而是一个被边界收纳的自地而上的生成事件。

优先判入 C 的情况：
- 主题明显带有破土、长出、升起、穿破、裂开、裂出、喷出、冒出、突刺、隆起、蔓生、爆长、喷涌等语义
- 用户想要的是“从地底出来”的事件感，而不是“从天上下来”的打击感
- 核心事件是“地下能量或实体向上生成并穿出地面”
- 例子：树木破土而出、冰川穿破地面、岩浆喷涌、晶簇突生、荆棘蔓生、石柱升起、骸骨冒出、地脉喷薄

C 的核心结构：
- 一个贴附在地面上的边界明确的技能容器
- 外边界清晰闭合，与纯黑背景明确分离
- 内部有承载地层、能量地表或受击地面
- 中央或局部有裂口、隆起点、喷涌核、破土源或生成源
- 某种主题意象从地面内部向上生成、突起、喷出、裂出或蔓生
- 不依赖复杂传统符文法阵成立

C 的视觉重点：
- 被边界收纳的地面生成事件
- 自下而上的力量方向
- 裂地、破土、拱起、突刺、喷涌、冒升、蔓生
- 明确的地层响应与局部结构破开感
- 贴地展开
- 独立技能资产感强
- 不场景化，不灾害化，不影视镜头化

不适合 C 的情况：
- 主题更像静态领域、池、境、域、海、湖、沼等区域本体，应优先考虑 A
- 主题更像一个中央唯一主体从高处垂直降临完成一次主命中，应优先考虑 B1
- 主题更像多个打击体连续从上到下覆盖受击区域，应优先考虑 B2
- 主题更像飞过、绕行、游弋、盘旋等轨迹事件，应优先考虑 D

路由原则：
优先根据主题所暗示的运动语法、事件结构和力的来源方向进行判断，而不是只看题材词。若主题核心是“从地面内部向上生成或破土喷涌”，优先考虑 C。
